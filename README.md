- CSS

## 💻 Projeto

Eternize é uma plataforma web que permite criar sites de homenagem para datas comemorativas como Dia das Mães, aniversários, Natal e muito mais. O usuário escolhe um template, personaliza com fotos e mensagens, e compartilha um link único — sem código, sem complicação.
